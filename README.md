## 網站分析摘要

這個頁面是一個「廟宇數位化管理系統（Temple Digital Management System）」的產品型 Landing Page。

核心特色模組大致為：

- Hero 區塊：  
  - 大標：AI 廟祝 / 數位化廟宇管理系統。  
  - 副標：強調「AI 智能解籤（AI divination explanation）」、「線上祈福（online blessing）」、「進香團管理（pilgrimage tour management）」、「電子點燈（digital lighting）」、「數位轉型（digital transformation）」。
  - 主要 CTA：「免費諮詢體驗（Free consultation）」按鈕，鏈往聯絡表單區。  

- 功能介紹區（Features section）：  
  - 區塊標題：系統特色（System Features）。  
  - 子模組：  
    - 會員管理（Member Management）：會員資料、活動參與紀錄、消費紀錄、個人化服務。
    - 活動管理（Event Management）：活動報名、通知推播、參與者管理、活動數據統計。
    - 線上服務（Online Services）：AI 解籤、線上祈福、商品購買、電子支付整合。
    - LINE 活動卡片推播（LINE Rich Message Promotion）：活動卡片、一鍵報名、群組通知、報名狀態更新。

- 系統介面展示（UI Showcase）：  
  - 幾個子段落：AI 智能解籤、LINE 智能推播、活動報名服務、數據營運中心（Dashboard）、會員服務整合（Member Service Integration），每一個有一小段說明，可搭配介面示意圖。  

- 體驗引導區（Experience section）：  
  - 引導加入官方 LINE，體驗推播、活動報名、會員管理等功能，有「加入好友（Add LINE friend）」按鈕。  

- 聯絡表單（Contact Form）：  
  - 欄位：廟宇名稱（Temple Name）、聯絡人姓名（Contact Person Name）、聯絡電話（Phone）、電子郵件（Email）、需求說明（Requirement Description）。

- Footer 區塊：  
  - 關於我們（About Us）：連結傳統與科技的描述。  
  - 功能特色（Features quick links）。  
  - 技術支援（Technical Support）：導入輔導、問題排除、功能更新、教育訓練。  
  - 版權宣告（Copyright）。

整體風格：SaaS 產品 Landing Page 結構明確，以「問題–解方–功能–介面示意–CTA–表單」的典型漏斗設計。
